# Ratchetron by @MichaelRelaxen 
- Fix a bug where after connecting/disconnecting too many times, the PS3 would run out of threads, and effectively softlock.
- Fix a bug where disconnecting Ratchetron at the wrong time (for instance not closing RaCMAN, then closing and re-connecting) would crash the game.
- These fixes should mitigate most of the console crashes people run into.
