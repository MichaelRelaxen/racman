Practice tool for boxxies in Ratchet & Clank 3 (NPEA00387).

It contains the following:
   An in-game display that shows which frame the crate is currently being sucked to.
   Current box speed.
   Ability to increase/decrease box speed
   Ability to automatically switch weapons after a specified amount of frames (frame request).
   Teleport to stored box.
   Save and load box.
   Respawns broken crates.

Hold down L3 button, then press:    
   Dpad Up/Down - Set frame request
   Dpad Left/Right - Set box speed     
   Select - Save box
   Start - Load box          
   R2 - Teleport to box
